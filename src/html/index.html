sportshub-{{version}} has the following routes:

Informational

Link Description
/ returns a list of all available routes
/version returns the version of the server
/info/leagues returns a list of all leagues
/info/sports returns a list of all sports

API

Link Input Output
/all none all events
/active none all active events
/sport/<sport> sport all events of a specific sport
/league/<league> league all events of a specific league
/id/<id> id a specific event
/team/home/<team> team all events where the home team is
/team/away/<team> team all events where the away team is
/team/<team> team all events where is either home or away
